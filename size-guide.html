AllChinaBuy size chart guide for spreadsheet finds

Learn how to read AllChinaBuy size charts, compare clothing measurements, check shoe insole length, and request warehouse measurement photos before shipping a haul.

Compare measurements with clothes you own

Do not rely only on S, M, L, or XL labels. For tops, compare shoulder width, chest, sleeve, and length. For pants, compare waist, rise, thigh, inseam, and leg opening. For shoes, compare insole length.

Watch for Asian sizing differences

Some AllChinaBuy spreadsheet finds use China-market sizing, which can run smaller than US or EU expectations. If a seller recommends sizing up, still confirm the actual chart and warehouse measurements.

Request measurement photos for risky items

Hoodies, jackets, wide pants, denim, and shoes are easier to judge with measurement photos. A listing image can look right while the fit is wrong, so measurements should decide whether the item stays in the haul.

Keep sizing notes while browsing finds

When comparing AllChinaBuy product links, keep notes on seller size charts, QC measurement photos, fit comments, and whether the item is oversized, cropped, slim, or true to measurement.
